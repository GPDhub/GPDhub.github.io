console.log

let(変数)

説明 letは、変数の宣言を差します。
変数名(name)にイコール後の値を入れ込みます。
左辺に右辺を代入しています。
使い方 let name = "John";
コンソール John

const(定数)

説明 constは、変数の宣言を差します。
使い方 const name = "John";
コンソール John

if

説明 ifは、条件分岐です。
使い方 const number = 12;
if (number > 10) {
console.log("numberは10より大きい値です");
}
コンソール numberは10より大きい値です

else

説明 elseは、条件分岐の再分岐です。
使い方 const number = 7;
if (number > 10) {
console.log("numberは10より大きい値です");
} else {
console.log("numberは10より小さい値です");
}
コンソール numberは10より小さい値です

複数の引数

説明 複数の引数（ひきすう）
使い方 const introduce = (name, age) => {
console.log(`私は${name}です`);
console.log(`私は${age}歳です`);
};

説明 呼び出し元で値を受け取れるようになります。returnの後にある関数内の処理は実行されません。

};
const sum = add(2, 3);
console.log(sum);
const add = (a, b) => {
console.log(a + b);
};
add(2, 3);

constructor(コンストラクタ)の中で、オブジェクトとして定義します。
this.state に直接代入してよい唯一の場所はコンストラクタです。
