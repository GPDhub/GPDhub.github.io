letは、変数の宣言を差します。
変数名(name)にイコール後の値を入れ込みます。

a === b ... aとbが等しい(trueを取得)

swich

説明

使い方

const color = "赤";
swich (color) {
case "赤";
console.log("ストップ);
break;

console.log("メダルはありません");
break;
}

while

説明

while文は、繰り返し処理を行うときに用います。
while以後の()内の条件式がtrueの間、{ }内の処理を繰り返すことができます。

使い方

let number = 1;
while (number <= 100) {
console.log(number);
number += 1;
}

for

説明

繰り返し処理を行う方法としてforを用います

使い方

for (let number = 1; number <= 100; number++) {
console.log(number);
}

オブジェクト

説明

プロパティと値がセットになってるデータを複数まとめて管理するのに用いられます。
{ プロパティ:値1, プロパティ:値2 }のように定義
プロパティと値セットで一つの要素とします。

使い方

for (let number = 1; number <= 100; number++) {
console.log(number);
}

重複オブジェクト

説明

プロパティの中にオブジェクトが挿入されている。
プロパティ:値,
プロパティ:値,
プロパティ: {
プロパティ:値,
プロパティ:値,
},
のように定義。｛}内がオブジェクト。

使い方

const character = {
name:"たけし",
age:14,
favorite: {

},

console.log(character.favorite.sports);

constructor(コンストラクタ)

コンストラクタは、this.state を直接代入する唯一の場所です。
他のすべてのメソッドでは、代わりに this.setState() を使う必要があります。

フォーム
state

ユーザーの動きに合わせて変わる値のこと。ボタンを押すとstateが変わり、表示される名前が変わる。
constructor(コンストラクタ)の中で、オブジェクトとして定義します。
this.state に直接代入してよい唯一の場所はコンストラクタです。
